Expose core WordPress content, media, taxonomy, migration, site health, and block editing capabilities through the WordPress Abilities API.

== Description ==

Filter Abilities registers WordPress Abilities API abilities that can be exposed to MCP-compatible tools through a WordPress MCP adapter.

The Lite edition includes the WordPress.org-safe core abilities:

* Content management
* Taxonomy management
* Media management
* Migration helpers
* Site health
* Block editing

The paid edition adds integrations for ACF, Yoast SEO, Gravity Forms, Filter AI, Redirection, PersonalizeWP, and WooCommerce Teams.

== Installation ==

1. Upload the `filter-abilities` folder to `/wp-content/plugins/`.
2. Activate Filter Abilities from the Plugins screen.
3. Install and configure a compatible WordPress MCP adapter to expose the abilities over MCP.
